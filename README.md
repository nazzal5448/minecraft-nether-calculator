# Minecraft Nether --> Overworld Calculator
Download the zip file, extract it, and load it unpacked in your browser. 

The official Extension is  available on:
[chrome store](https://chromewebstore.google.com/detail/nether-calculator/mpoccmenmjcplbbmmahbogibbogjapci)
[Mozilla Firefox store](https://addons.mozilla.org/en-US/firefox/addon/nether-calculator/)
[Edge Addons store](https://microsoftedge.microsoft.com/addons/detail/nether-calculator/ahmngilppdffipiiddfmhefecnogbjak).
